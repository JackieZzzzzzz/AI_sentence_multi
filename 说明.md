# 📊 企业年报 (MD&A) 人工智能语料抽取流水线

本项目是一个基于自然语言处理 (NLP) 的自动化流水线，专为处理大规模企业年报（MD&A部分）文本而设计。其核心目标是通过**分词、Word2Vec词向量训练、同义词正则替换及长短句精准切割**，最终从海量文本中自动化提取出与“人工智能”相关的核心语句，并生成结构化的 CSV 统计结果。

---

## ✨ 核心特性 (Features)
- 🚀 **极速多进程架构：** 针对分词和分句这种高 I/O、高 CPU 消耗的任务，采用 `ProcessPoolExecutor` 解锁多核算力，相比单线程速度提升 3-5 倍。
- 💾 **断点续传保护：** 自动识别已处理的文件。如果程序意外中断，下次运行会自动跳过已完成的文件，拒绝重复劳动。
- 🧪 **一键测试模式：** 内置 `TEST_MODE` 开关，允许以少量样本（如 5-10 个文件）快速跑通完整流水线，方便调试代码逻辑。
- ⚙️ **高度模块化：** 采用“集中配置 + 独立模块”设计，按年份隔离数据，扩展性极强。

## 📂 项目结构 (Project Structure)

建议将项目按照以下结构进行组织：

```text
AIWashing/
│
├── config.py                     # 全局配置中心（路径、年份、测试开关）
├── main.py                       # 🚀 唯一启动入口（流水线总控）
├── module1_tokenizer.py          # 模块1：多进程分词与规范化
├── module2_w2v.py                # 模块2：Word2Vec 模型训练与扩词
├── module3_sentence_parser.py    # 模块3：多进程正则替换与分句
├── module4_extractor.py          # 模块4：目标语句抽取与数据导出
│
│── AI_dictionary_merged.txt  # [用户提供] 自定义实体词典
│── synonym.txt               # [用户提供] 同义词映射表
└── 2024/                         # 工作目录 (以年份命名)
    ├── 2024_MD&A/                # [用户提供] 存放原始年报 .txt 文件的文件夹
    │
    ├── 2024_token/               # [自动生成] 分词后的文本存放处
    ├── 2024_replace/             # [自动生成] 同义词替换后的文本存放处（内部中间件，代码已融合入模块3）
    ├── 2024_sentence/            # [自动生成] 彻底清洗并按行分句后的文本存放处
    ├── metadata_2024.csv         # [自动生成] 全局文件元数据索引表
    ├── Model2024.model           # [自动生成] 训练好的 Word2Vec 模型
    ├── result2024.txt            # [自动生成] 模型提取出的 Top100 人工智能相关词
    ├── step3_ai_sentences_detail_2024.csv # [最终输出] 包含AI关键词的句子明细表
    └── step3_report_stats_2024.csv        # [最终输出] 每份报告的总句子数统计表
